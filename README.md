# School Management System

A full-stack School Management System built using React, FastAPI, SQLite, and SQLAlchemy.

## Features

- Add Student
- View Student List
- Edit Student Details
- Delete Student Records
- Search Students
- Dashboard with Student Count

## Tech Stack

### Frontend
- React
- React Router
- Axios
- Bootstrap

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic

## Future Enhancements

- Authentication
- AI Student Assistant
- Student Performance Analytics
